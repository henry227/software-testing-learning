## 💻 实战项目：注册登录系统测试

### 项目概述
一个完整的用户注册登录系统，用于测试练习。

### 系统功能
1. **用户注册**
   - 用户名、邮箱、密码输入
   - 密码确认
   - 验证码验证
   - 注册协议同意

2. **用户登录**
   - 用户名/邮箱登录
   - 密码输入
   - 记住登录状态
   - 忘记密码功能

3. **用户管理**
   - 个人信息查看
   - 密码修改
   - 账号安全设置

### 测试重点
1. **功能测试**：各项功能是否正常
2. **边界测试**：输入边界值测试
3. **安全测试**：密码安全、SQL注入防护
4. **兼容性测试**：不同浏览器/设备测试
5. **性能测试**：并发用户登录测试
