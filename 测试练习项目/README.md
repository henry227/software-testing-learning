# 注册登录系统测试练习项目

这是一个用于软件测试练习的简单注册登录系统。

## 项目结构

```
测试练习项目/
├── frontend/          # 前端页面
│   ├── index.html     # 首页
│   ├── register.html  # 注册页面
│   ├── login.html     # 登录页面
│   └── profile.html   # 个人中心页面
├── backend/           # 后端模拟API
│   └── api.js         # 模拟API服务
├── test-cases/        # 测试用例
│   ├── 功能测试.md
│   ├── 边界测试.md
│   └── 安全测试.md
└── automation/        # 自动化测试脚本
    ├── selenium-test.py
    └── api-test.py
```

## 快速开始

### 1. 启动前端服务
```bash
# 使用Python简单HTTP服务器
cd frontend
python -m http.server 8000
```

### 2. 启动后端模拟API
```bash
cd backend
node api.js
```

### 3. 访问系统
- 首页: http://localhost:8000
- 注册页: http://localhost:8000/register.html
- 登录页: http://localhost:8000/login.html

## 系统功能

### 注册功能
- 用户名输入（4-20位字母数字）
- 邮箱输入（有效邮箱格式）
- 密码输入（8-20位，包含大小写字母和数字）
- 确认密码
- 验证码输入（模拟）
- 用户协议同意

### 登录功能
- 用户名/邮箱登录
- 密码输入
- 记住我功能
- 忘记密码链接

### 个人中心
- 个人信息展示
- 修改用户名
- 修改密码
- 安全设置

## 测试任务

### 第一阶段：手动测试
1. 功能测试 - 验证所有功能是否正常
2. 边界测试 - 测试输入边界条件
3. 错误处理测试 - 测试异常情况处理

### 第二阶段：自动化测试
1. Selenium自动化测试
2. API接口测试
3. 性能测试

### 第三阶段：测试报告
1. 缺陷报告编写
2. 测试总结报告
3. 改进建议
